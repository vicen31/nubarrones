Instalación con Docker

Las imágenes de Docker que continen carácetrísticas disponibles bajo licencia Apache 2.0 se pueden descargar desde aquí. Hay que buscar aquellas con extensíon -oss.

  • Elasticsearch

docker pull docker.elastic.co/elasticsearch/elasticsearch-oss:7.1.1

docker run -p 9200:9200 -p 9300:9300 -e "discovery.type=single-node" docker.elastic.co/elasticsearch/elasticsearch-oss:7.1.1

  • Kibana

docker pull docker.elastic.co/kibana/kibana-oss:7.1.1

  • Logstash

docker pull docker.elastic.co/logstash/logstash-oss:7.1.1

Elasticsearch - arranque y parada

sudo systemctl start elasticsearch.service

sudo systemctl stop elasticsearch.service

curl -X GET "localhost:9200/"

Ejecutable:

sudo service elasticsearch start

NO FUNCIONA :cd /usr/share/elasticsearch/bin && ./elasticsearch

http://localhost:9200/_cat

Elasticsearch - Directorios

Instalación: /usr/share/elasticsearch

Configuracion: /etc/elasticsearch

Logs: /var/log/elasticsearch

Añadir usuario al grupo elasticsearch: sudo usermod -a -G elasticsearch vicente

Kibana - arranque y parada

sudo systemctl start kibana.service

sudo systemctl stop kibana.service

http://localhost.com:5601
